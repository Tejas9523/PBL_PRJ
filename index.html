Virtual Mouse using Hand Gestures


A virtual mouse is a computer input device that allows users to control a cursor on a display screen without the use of a physical mouse. Virtual mice can be controlled by a variety of input methods, such as hand gestures, head movements, or voice commands.

This project will develop a virtual mouse that is controlled by hand gestures. The virtual mouse will be implemented using a webcam and a computer vision algorithm. The computer vision algorithm will track the user's hand movements and translate them into mouse movements
